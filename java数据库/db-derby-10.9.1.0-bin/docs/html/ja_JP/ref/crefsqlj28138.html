数値型の概要

数値型には以下の型があります。それぞれの型は異なるサイズの記憶容量を持ちます。
  • 整数
    • SMALLINT (2 bytes)
    • INTEGER (4 bytes)
    • BIGINT (8 bytes)
  • 近似数または浮動小数点数
    • REAL (4 bytes)
    • DOUBLE PRECISION (8 bytes)
    • FLOAT (DOUBLE PRECISIONあるいはREALの別名)
  • 正確な数
    • DECIMAL (精度に応じた記録)
    • NUMERIC (DECIMALの別名)
