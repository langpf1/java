構文

DROP FUNCTION 関数名

破棄する関数を指定します。指定した関数名の関数がスキーマにて一つのみ見つかった場合に有効です。指定された関数はいくつ引数を持っていてもかまいません。指定された名前の関数がスキーマに見つからなかった場合はエラーとなります。(SQLSTATE 42704) 指定された名前やスキーマの関数が複数見つかった場合もエラーとなります。
