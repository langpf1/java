DerbyのサポートするSQL-92の機能

SQL-92のサポートには4つのレベルがあります。
  • SQL92E

    初歩的

  • SQL92T

    過渡的　FIPS 127-2という発行物にてNISTが定義

  • SQL92I

    中級

  • SQL92F

    完全
