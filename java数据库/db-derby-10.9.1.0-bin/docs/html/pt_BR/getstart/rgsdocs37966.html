Convenções da documentação

Esta seção descreve a terminologia, sintaxe e convenções tipográficas da documentação do Derby.

  • Terminologia
    A documentação do Derby utiliza o termo especializado ambiente para descrever o método utilizado pelo aplicativo para interagir com o Derby.
  • Sintaxe SQL
    A sintaxe SQL é apresentada em notação BNF modificada.
  • Convenções tipográficas
    Esta documentação utiliza algumas convenções tipográficas para realçar elementos da linguagem SQL, comandos do sistema operacional e da linguagem de programação Java.
