Restrições dos pontos de salvamento

O Derby não suporta pontos de salvamento em gatilhos.

O Derby não libera bloqueios como parte de desfazer até o ponto de salvamento.
Tabela 1. Métodos de Savepoint do JDBC 3.0
Retorna Assinatura Notas da implementação
int getSavepointId() Lança uma SQLException se for um ponto de salvamento com nome. Obtém o ID gerado para o ponto de salvamento que este objeto Savepoint representa.
String getSavepointName() Lança uma SQLException se for um ponto de salvamento sem nome. Obtém o nome do ponto de salvamento que este objeto Savepoint representa.
