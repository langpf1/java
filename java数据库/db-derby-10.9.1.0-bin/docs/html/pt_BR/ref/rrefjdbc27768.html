Sintaxe de escape do JDBC

O Derby interpreta a sintaxe de escape do JDBC para datas como como equivalente a sintaxe SQL para datas.

Sintaxe

{d 'yyyy-mm-dd'}

Equivalente a

DATE('yyyy-mm-dd')

VALUES {d '1999-01-09'}
