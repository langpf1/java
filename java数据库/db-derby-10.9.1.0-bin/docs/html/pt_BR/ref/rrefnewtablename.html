novo-nome-da-tabela

O novo-nome-da-tabela representa uma tabela renomeada. Não é permitido qualificar o novo-nome-da-tabela com o nome-do-esquema.

Sintaxe

IdentificadorSQL92

Exemplo

-- VÔOS_DISPONÍVEIS é o novo-nome-da-tabela que não inclui o nome-do-esquema

RENAME TABLE VÔOS_DISPONIBILIDADE TO VÔOS_DISPONÍVEIS
