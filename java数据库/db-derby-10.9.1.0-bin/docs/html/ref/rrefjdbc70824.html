JDBC escape keyword for call statements

This syntax is supported for a java.sql.Statement and a java.sql.PreparedStatement in addition to a CallableStatement.

Syntax

{call statement }

Example

-- Call a Java procedure
{ call TOURS.BOOK_TOUR(?, ?) }
